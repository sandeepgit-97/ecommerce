# ecommerce
This is a Ecommerce platform where you can access huge quantity of branded men's wearing stuffs.
To Run this site you first need to install Django in your vs code. 
First create an Virtual Environmnet "python -m venv myworld"
Then you have to activate the environment, by typing this command: "Virtual_Environment_name\Scripts\activate" (windows) "source virtual_env_name/bin/activate" (Unix/MacOS)
Now install Django in your vs-code "python -m pip install Django".

Now run the server :"python manage.py runserver".
